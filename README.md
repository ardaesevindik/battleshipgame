A smooth, terminal-based 1D Battleship game written in Python. Built entirely with standard libraries (`random`, `os`, `time`), this project focuses on a fluid command-line experience. It features a letter-by-letter typing intro, a dynamic percentage-based loading bar, and seamless screen transitions using ANSI escape codes. 

**The Mission:** The system hides a ship within a 5-cell grid (`[1] [2] [3] [4] [5]`). You have exactly 3 attempts to fire at the correct coordinates and sink it before you run out of ammo.
